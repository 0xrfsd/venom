# Bot to test functions

> Need to do _npm install_ in the main venom folder and _npm start_ here

| Message                  | Return                  |
| ------------------------ | ----------------------- |
| `!ping`                  | pong                    |
| `!ping reply`            | pong like reply         |
| `!chats`                 | total chats opened      |
| `!info`                  | infos about connection  |
| `!sendto <number> <msg>` | send msg to some number |
| `!pin <true              |                         | false>` | freeze this chat |
| `!typing <true           |                         | false>` | change state chat to typing |
| `!ChatState < 0: Typing  |                         | 1: Recording |  | 2: Paused >` | change state chat to options |
